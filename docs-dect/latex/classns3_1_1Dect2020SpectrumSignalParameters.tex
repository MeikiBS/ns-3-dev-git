\doxysubsection{Detailed Description}
Signal parameters used by the DECT-\/2020 PHY layer in Spectrum\+Channel transmissions. 

This class inherits from ns3\+::\+Spectrum\+Signal\+Parameters and encapsulates\+:
\begin{DoxyItemize}
\item The packet to be transmitted over the spectrum channel
\item The DECT-\/specific channel ID on which the transmission takes place
\end{DoxyItemize}

These parameters are passed to ns-\/3\textquotesingle{}s Spectrum\+Channel\+::\+Start\+Tx() method and received by other PHY entities through Start\+Rx().

This structure allows DECT-\/2020 NR devices to embed channel-\/specific transmission metadata into the physical layer for RSSI calculation, collision detection, and subslot-\/based scheduling.
