\doxysubsection{Detailed Description}
Represents the PHY Control Field Type 2 (Format 001) for DECT-\/2020 NR. 

This extended header includes\+:
\begin{DoxyItemize}
\item All fields from Type 1 (e.\+g., packet length, transmitter ID, power)
\item Receiver identity
\item Number of spatial streams
\item Feedback format and content (e.\+g., CQI, HARQ)
\end{DoxyItemize}

Based on ETSI TS 103 636-\/4 V2.\+1.\+1, Table 6.\+2.\+1-\/2a.
