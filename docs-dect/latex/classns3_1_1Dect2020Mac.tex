\doxysubsection{Detailed Description}
DECT-\/2020 NR MAC layer implementation. 

Represents the Medium Access Control layer for a DECT-\/2020 NR node. Manages transmission/reception of MAC PDUs, beaconing, association, and MAC-\/\+PHY interaction.
