\doxysubsection{Detailed Description}
Definition of physical layer control field headers (Type 1 and Type 2) for DECT-\/2020 NR. 

This header defines two classes that represent the physical layer control fields used in DECT-\/2020 NR, based on ETSI TS 103 636-\/4 Section 6.\+2.\+1\+:
\begin{DoxyItemize}
\item Dect2020\+PHYControl\+Field\+Type1\+: for basic one-\/way transmissions (Format 000)
\item Dect2020\+PHYControl\+Field\+Type2\+: for enhanced transmissions with feedback and spatial streams (Format 001)
\end{DoxyItemize}

These headers are serialized and prepended to PHY layer packets for transmission via Spectrum\+Channel.
