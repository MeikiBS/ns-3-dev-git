\doxysubsection{Detailed Description}
Describes parameters for a DECT band. 

This structure defines the properties of a DECT band as specified in ETSI TS 103 636-\/2 V1.\+5.\+1, Table 5.\+4.\+2-\/1. It includes the frequency range and step size used to compute valid logical channel numbers.
