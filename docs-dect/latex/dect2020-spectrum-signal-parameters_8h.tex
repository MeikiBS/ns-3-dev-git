\doxysubsection{Detailed Description}
Defines signal parameter structure for DECT-\/2020 NR PHY layer transmissions. 

This header extends ns-\/3\textquotesingle{}s Spectrum\+Signal\+Parameters to include DECT-\/specific transmission attributes such as the currently used channel ID and the pointer to the transmitted packet.

It is used by the Dect2020\+Phy layer to transmit packets via ns-\/3\textquotesingle{}s Spectrum\+Channel framework.
