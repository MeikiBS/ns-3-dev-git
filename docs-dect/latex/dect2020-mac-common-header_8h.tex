\doxysubsection{Detailed Description}
Implements MAC Common Headers according to ETSI TS 103 636-\/4, Section 6.\+3.\+3. 

This file defines the common header structures used by DECT-\/2020 NR MAC layer messages. Currently implemented Headers are Beacon and Unicast header.
