void ns3::Dect2020NetDevice::SetTerminationPointType ( TerminationPointType  tpm)

Sets the termination point type of the device.

Note: In typical DECT implementations, the termination point type (PT or FT) would be dynamically determined by the device. However, for simplicity and initial testing purposes, the type is set manually during device creation in this simulation.
