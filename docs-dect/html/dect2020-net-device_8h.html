Detailed Description

DECT-2020 NetDevice implementation for ns-3.

This file contains the implementation of the ns-3 NetDevice abstraction for DECT-2020 NR. The class encapsulates MAC and PHY components, and provides bootstrapping capabilities and core NetDevice API functionality.
